November 28th, 2025

  • Recieved the developed project, went through the code creating a full class diagram of the entire game, felt neccessary to layout the game in a human-friendly format so implementation begins smoother

December 5th, 2025

  • Added new classes to the diagram that have been added or plan to be added, dependencies that will be required have been highlighted. Additionally, cleaned up dependencies making the diagram look more clean and readable
  • Achievements - example achievements, being challenges for the player to achieve, have been added to the diagram as well as an example contianing methods that it may have

  • Achievement example classes have been removed due to them being generic and having almost identical proporties, however all dependencies are now displayed from the abstract class achievement
  • More generic events have been added with their dependencies displayed, an unaccounted for win screen has now been addressed
