Manual Testing

Manual tests for the game

  1. Open the game and verify the main menu loads correctly.
  2. Navigate through all menu options (Play, How to Play, Settings, Leaderboard, Achievements, Credits, Quit).
  3. Confirm that selecting each option performs the intended action.
  4. Check all UI elements (Time, score, event counter, text popups) for alignment and clarity.
  5. Check that text is readable and colours contrast properly with backgrounds.
  6. Play the game and verify all sprites and textures render correctly.
